#練習問題 8.4 関数find を第三の仮引数を持つように修正せよ。この第三仮引数は探索を始めるインデックスの値である。
	def find(word, letter, index):
		while index < len(word):
			if word[index] == letter:
				return index
			index = index + 1
		return -1
